Before Using Application

  • Before using the application, you need to accept permission to utilise the full functionality of the application
  • If these permission are not accepted, then users cannot continue on to the application

Registration

  • To register and account on the application, user should type in thier desired username and password for the account
  • username and password fields cannot be empty
  • If username is available then the account and the password in the both the password fields match, the account will be successfully created
  • Once account registration is complete, then the user is automatically logged in

Logging In

  • User should input thier correct username and password inorder to successfully log into thier account
  • Again username and password fields cannot be empty
  • Login will be unsuccessful if the password inputted does not match password inputted upon account registration

Home Screen

  • Once successfully logged in, users can access all the functionality of the application
  • Users can log out of their account by clicking button in the top right hand corner
  • Users can access a settings page by clicking the settings button in the top right hand corner
  • Users can choose to play against a computer by clicking on single player button
  • Users can choose to play against another player by clicking on two player button
  • Users find a guide on how to play Tic-Tac-Toe by clicking the tutorial button
  • Users can view the leaderboard rankings for all users that have utilised the application
  • Through out the application, users can swipe on right edge of screen to return to the previous page they were on

Difficulty Page

  • Users can choose how difficulty the computer can be from three options: easy, medium and hard
  • Once option is choosen, the user can start the game

Settings Page

  • Users can see thier overall data from scores against computer
  • Data includes the player score and computer score on each difficulty available
  • Users can also share the application to other people by clicking the share button at top right hand corner
  • Users can access a settings page by clicking the settings button in the top right hand corner

Leaderboard Page

  • Users can view thier leaderboard ranking in comparison to other users
  • Users can filter the leaderboard with the number of wins and the win percentage
  • These filters can be combined with the difficulty filter to give a more accurate ranking

Single Player Functionality Page

  • User can play against the computer by clicking button on field where they wish to play
  • Once user has played, the computer will respond with is own move. This may take a second to register
  • Once the game has ended, the loser of the game will get to go first
  • Score counter is at the top left corner of the screen
  • If user does not make thier move in 30 seconds, a notification will remind them to finish thier game
  • User can reset board and score counter by clicking the reset button
  • User can return to the home screen by pressing the back button
  • User can return to difficulty page by utilising the edge swipe gesture
  • User can also return to thier previous game state when the screen is changed or when the app is closed

Two Player Functionality Page

  • User can make thier move by clicking button on field where they wish to play
  • Once user has played, the other player can respond with thier own move. This may take a second to register
  • Once the game has ended, the loser of the game will get to go first in the next round
  • Score counter is at the top left corner of the screen to keep track of the score for this session
  • User can reset board and score counter by clicking the reset button
  • User can return to the home screen by pressing the back button
